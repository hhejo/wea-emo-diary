## 1. 소개

**Weather Emotion Diary**

주제

- `Scratch`를 이용한 모바일 애플리케이션
- 오늘의 기분, 날씨, 평가를 클릭해 저장

## 2. 기술 스택

`Scratch`

## 3. 느낀 점

- `Scratch`는 아이들 교육용인 줄 알았는데, 쓸만한 앱을 간단하고 편하게 만들 수 있었음

## 4. 기능 (페이지 구성)

1. 종합 상태창
2. 날씨 버튼 3가지, 기분 버튼 3가지, 평가 버튼 2가지
3. 기록, 목록 버튼
4. 목록의 아이템을 클릭하면 삭제
